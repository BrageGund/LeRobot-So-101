## LeRobot-So-101

# Overview

This project documents the assembly, programming, and testing of the LeRobot SO-101 robotic arm. The goal of this project is to gain hands-on experience with robotics, combining mechanical design, embedded systems, and Python-based control.

The robot arm is assembled using 3D printed components and controlled with a microcontroller, allowing experimentation with motion control, kinematics, and sensor integration.

This project is part of my broader effort to develop practical robotics and automation skills alongside my mechanical engineering studies.

# Project Goals

- Assemble and understand the mechanical design of a robotic arm
- Interface microcontrollers with motors and sensors
- Control robotic motion using Python
- Explore robotics concepts such as joint control and basic kinematics
- Build a foundation for future robotics and automation projects
